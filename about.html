About This Web Application

Project Description ▼

This web application, named CCAPDEV MCO1, was developed using various NPM packages and third-party libraries to ensure a seamless and efficient user experience.

Dependencies ▼
  • bcrypt: Password hashing library
  • cookie: Cookie management utility
  • cookie-parser: Middleware for parsing cookies
  • dotenv: Environment variable management
  • express: Web framework for Node.js
  • express-session: Session management middleware
  • mongoose: MongoDB object modeling tool
  • multer: Middleware for handling file uploads
  • parser: General-purpose data parser
DevDependencies ▼
  • nodemon: Development tool for automatically restarting the server
